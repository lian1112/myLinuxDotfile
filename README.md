# My Linux Dotfiles

個人 Linux 配置檔備份

## 包含檔案

- `vimrc` - Vim 編輯器配置
- `zshrc` - Zsh Shell 配置

## 安裝方式
```bash
git clone git@github.com:lian1112/myLinuxDotfile.git ~/myLinuxDotfile
cd ~/myLinuxDotfile
./install.sh
```

## 更新配置

修改檔案後直接 commit 並 push：
```bash
git add .
git commit -m "[FEAT] 更新配置"
git push
```
